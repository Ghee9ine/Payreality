# PayReality

**Independent Control Validation for Internal Audit**

PayReality exposes Identity Hallucination in ERP systems by matching actual payee names against approved vendor masters using advanced fuzzy matching.

## Features

- Advanced fuzzy matching with multiple strategies
- Professional PDF reports with executive summaries
- Control Entropy Score calculation
- Configurable matching thresholds
- Audit logging
- Multi-file support (CSV, Excel)
